# Choices-esm.js 

An ES Module build of [Choices.js](https://github.com/jshjohnson/Choices#readme) TypeScript via Rollup.

## Use

```js
import Choices, {templates} from 'choices-esm';
```

## Updated Build

1. Update dependencies in `package.json`, particularly `"choices.js"`
2. `npm install`
3. `npm build`
4. `npm publish`
